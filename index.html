What is verifier?

verifier is a tool for code verification of isosurface extraction algorithms. Isosurface extraction techniques are very useful in the context of Computations Science & Engineering and Medical Imaging fields. Whle there are plenty of code freely available for extracting isosurfaces from volume data, there are not many tools that can be used to check whether the results of the algorithm preserves basic properties of isosurfaces.verified can help verify the code.
